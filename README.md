# Metro Planner Manchester


Metro planner that takes in a csv data which should be placed in the same folder and creates a GUI to help you plan out a journey with constraints such as fastest journey time and least amount of line changes.

# To compile: 
javac *.java
(then)
java MetroGui

# To update javadocs: 
javadoc -d javaDocs MetroGui.java SearchWithSpec.java MetroGraph.java
